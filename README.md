# 使用方法
把可话的导出文件解压（默认名为我的动态），替换main同级目录下“我的动态”文件夹。  
打开main.py 修改前几行配置,示例：  

```
EMAIL = "123456@qq.com"  
PASSWORD = "password"  
START_DATE_STR = "2021-09-01"  
END_DATE_STR = "2025-12-10"
```
注意账号要填写邮箱。如果账号是qq则请自动补全为qq邮箱填写。  

安装需要的库
```
pip install requests
```
运行main.py即可。
